Here’s how you can do both in **Tailwind CSS**. I’ll break it down step by step so you can recreate it yourself.

### **1. Image background with text overlay**

You want something like a hero section where the image is behind text.

```html
<div class="relative w-full h-64 bg-cover bg-center" style="background-image: url('your-image.jpg');">
  <div class="absolute inset-0 bg-black bg-opacity-40 flex items-center justify-center">
    <h1 class="text-white text-3xl font-bold">Your Overlay Text</h1>
  </div>
</div>
```

**Explanation:**

* `relative` on the container → so the overlay can be positioned absolutely.
* `bg-cover bg-center` → makes the image cover the div and centered.
* `absolute inset-0` → overlay fills the container.
* `bg-black bg-opacity-40` → semi-transparent overlay for contrast.
* `flex items-center justify-center` → centers text horizontally and vertically.
* `text-white` → ensures text stands out.

---

### **2. Image with hover play icon and zoom effect**

You can make this with **group hover** in Tailwind:

```html
<div class="relative w-64 h-40 overflow-hidden rounded-lg group cursor-pointer">
  <img src="your-image.jpg" class="w-full h-full object-cover transition-transform duration-300 group-hover:scale-105" />
  
  <!-- Play icon overlay -->
  <div class="absolute inset-0 flex items-center justify-center opacity-0 group-hover:opacity-100 transition-opacity duration-300">
    <svg xmlns="http://www.w3.org/2000/svg" class="h-12 w-12 text-white" fill="none" viewBox="0 0 24 24" stroke="currentColor">
      <path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M14.752 11.168l-6.518-3.76A1 1 0 007 8.303v7.394a1 1 0 001.234.97l6.518-1.88a1 1 0 00.75-.97V12.14a1 1 0 00-.75-.972z" />
    </svg>
  </div>
</div>
```

**Explanation:**

* `group` → lets child elements respond to hover on the parent.
* `transition-transform duration-300` + `group-hover:scale-105` → subtle zoom on hover.
* `absolute inset-0 flex items-center justify-center` → centers the play icon.
* `opacity-0 group-hover:opacity-100 transition-opacity duration-300` → fades in the icon on hover.
* `overflow-hidden` → prevents image from overflowing during zoom.

✅ Result:

* Normal view → image only.

## **Step 1: Fetch Trending Movies**

TMDB Trending Movies endpoint:

```http
GET https://api.themoviedb.org/3/trending/movie/week?api_key=YOUR_API_KEY
```

In React, you can fetch it like this:

```javascript
import { useEffect, useState } from "react";

function TrendingMovies() {
  const [movies, setMovies] = useState([]);

  useEffect(() => {
    fetch("https://api.themoviedb.org/3/trending/movie/week?api_key=YOUR_API_KEY")
      .then(res => res.json())
      .then(data => setMovies(data.results))
      .catch(err => console.error(err));
  }, []);

  return (
    <div>
      <h2>Trending Movies</h2>
      <div className="movie-grid">
        {movies.map(movie => (
          <div key={movie.id}>
            <img 
              src={`https://image.tmdb.org/t/p/w500${movie.poster_path}`} 
              alt={movie.title} 
            />
            <p>{movie.title}</p>
          </div>
        ))}
      </div>
    </div>
  );
}

export default TrendingMovies;
```

✅ What’s happening:

1. `useEffect` fetches the trending movies **once on component mount**.
2. `data.results` contains an array of movies.
3. `map` renders each movie in a grid.
4. Poster URLs need `https://image.tmdb.org/t/p/w500` + `poster_path`.

function MovieDetails({ movieId }) {

### **1. Image with text overlay**

* Use `relative` on the container.
* Image inside.
* Text inside a `absolute` div over it.

```html
<div class="relative w-80 h-48">
  <img src="movie.jpg" alt="Movie" class="w-full h-full object-cover">
  <div class="absolute inset-0 flex items-end p-4 bg-gradient-to-t from-black/60 to-transparent text-white">
    <h2 class="text-lg font-bold">Movie Title</h2>
  </div>
</div>
```

**Explanation:**

* `relative` on container → needed for positioning overlay absolutely.
* `absolute inset-0` → overlay covers the entire image.
* `bg-gradient-to-t from-black/60 to-transparent` → subtle dark overlay at bottom for text readability.
* `flex items-end p-4` → positions text at bottom with some padding.

---

### **2. Hover effect with play icon and zoom**

* Wrap image and overlay in a `group`.
* Use `group-hover:` for hover effects.
* Animate `scale` for subtle zoom.
* Add overlay for dark effect and a centered play icon.

```html
<div class="relative w-80 h-48 overflow-hidden rounded-lg group cursor-pointer">
  <img src="movie.jpg" alt="Movie" class="w-full h-full object-cover transition-transform duration-500 group-hover:scale-105">
  
  <!-- Dark overlay on hover -->
  <div class="absolute inset-0 bg-black/30 opacity-0 transition-opacity duration-500 group-hover:opacity-100"></div>

  <!-- Play icon -->
  <div class="absolute inset-0 flex items-center justify-center opacity-0 transition-opacity duration-500 group-hover:opacity-100 text-white text-4xl">
    ▶️
  </div>

  <!-- Optional text overlay at bottom -->
  <div class="absolute bottom-0 w-full p-4 bg-gradient-to-t from-black/60 to-transparent text-white">
    <h2 class="text-lg font-bold">Movie Title</h2>
  </div>
</div>
```

**Explanation:**

* `group` on container → allows child hover effects.
* `transition-transform` & `group-hover:scale-105` → smooth subtle zoom on hover.
* `bg-black/30` with `opacity-0 → opacity-100` → subtle dark overlay on hover.
* `opacity-0 → opacity-100` on play icon → fades in smoothly.

---

### ✅ **Tips for realism**

1. Use `overflow-hidden` to prevent the image from overflowing when zooming.
2. `rounded-lg` or similar → smooth edges like Netflix or Disney+ cards.
3. `transition-all` or specific transitions make animations smoother.
4. You can replace the ▶️ with an SVG for a more professional play button.
